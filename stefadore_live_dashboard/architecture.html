How To Read It

GitHub Stores

GitHub is the versioned home for the dashboard code. When code is pushed there, Render can redeploy it.

Render Runs

Render hosts the public dashboard. It runs the Python server so your phone can open it anywhere.

YouTube Supplies

YouTube is the data source. The bundled snapshot works immediately; an API key makes live refreshes more reliable.
